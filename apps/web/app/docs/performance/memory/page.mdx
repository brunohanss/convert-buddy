# Memory

Streaming conversion keeps memory usage predictable by chunking input and output.

## Minimal example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { ConvertBuddy } from "convert-buddy-js";

const buddy = new ConvertBuddy({ memoryLimitMb: 128 });
`,
  }}
/>


## Advanced example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { ConvertBuddy } from "convert-buddy-js";

const buddy = new ConvertBuddy({
  memoryLimitMb: 256,
  onProgress: (stats) => console.log(stats.memoryMb)
});
`,
  }}
/>
