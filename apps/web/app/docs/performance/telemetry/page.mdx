# Telemetry

Telemetry reports throughput, memory, elapsed time, and bytes in/out at fixed intervals.

## Minimal example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { ConvertBuddy } from "convert-buddy-js";

const buddy = new ConvertBuddy({
  onProgress: (stats) => {
    console.log(stats.throughputMbPerSec);
    console.log(stats.memoryMb);
    console.log(stats.elapsedMs);
  }
});
`,
  }}
/>


## Advanced example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { ConvertBuddy } from "convert-buddy-js";

const buddy = new ConvertBuddy({
  onProgress: (stats) => {
    analytics.track("throughput", stats.throughputMbPerSec);
  },
  progressIntervalMs: 500
});
`,
  }}
/>
