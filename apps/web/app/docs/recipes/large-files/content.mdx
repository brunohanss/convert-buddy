# Large files

For multi-GB inputs, rely on streaming and telemetry to keep memory predictable.

## Minimal example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { ConvertBuddy } from "convert-buddy-js";

const buddy = new ConvertBuddy({ memoryLimitMb: 256 });
`,
  }}
/>
