# XML

XML is parsed with record element detection and streaming node extraction.

## Minimal example

<SandpackExample
  template="vanilla-ts"

`,
  }}
/>


## Advanced example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{

import { convertToString } from "convert-buddy-js";

const output = await convertToString(file, {
  inputFormat: "xml",
  outputFormat: "json",
  xml: { recordElement: "row" }
});
`,
  }}
/>
