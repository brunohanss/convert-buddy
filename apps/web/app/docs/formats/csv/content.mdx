# CSV

CSV is parsed with streaming delimiters, header detection, and schema inference.

## Minimal example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { convertToString } from "convert-buddy-js";

const output = await convertToString(file, { inputFormat: "csv", outputFormat: "json" });
`,
  }}
/>


## Advanced example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { convertToString } from "convert-buddy-js";

const output = await convertToString(file, {
  inputFormat: "csv",
  outputFormat: "json",
  csv: { delimiter: ",", hasHeaders: true }
});
`,
  }}
/>
