# NDJSON

NDJSON is processed line-by-line for deterministic streaming conversion.

## Minimal example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { convertToString } from "convert-buddy-js";

const output = await convertToString(file, { inputFormat: "ndjson", outputFormat: "json" });
`,
  }}
/>


## Advanced example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { convertToString } from "convert-buddy-js";

const output = await convertToString(file, {
  inputFormat: "ndjson",
  outputFormat: "json",
  ndjson: { allowTrailingComma: true }
});
`,
  }}
/>
