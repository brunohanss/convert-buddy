# Detect structure

Structure detection extracts fields, record element names, and schema hints.

## Minimal example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { detectStructure } from "convert-buddy-js";

const structure = await detectStructure(stream);
`,
  }}
/>


## Advanced example

<SandpackExample

  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { detectStructure } from "convert-buddy-js";

  }}
/>
