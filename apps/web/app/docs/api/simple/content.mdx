# Simple API

The simple API converts a file or stream in one call. It is ideal for UI flows and scripting.

## Minimal example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { convertToString } from "convert-buddy-js";

const output = await convertToString(file, { outputFormat: "json" });
`,
  }}
/>


## Advanced example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { convertToString } from "convert-buddy-js";

const output = await convertToString(file, {
  inputFormat: "csv",
  outputFormat: "json",
  transform: (r) => ({ ...r, channel: "web" })
});
`,
  }}
/>
