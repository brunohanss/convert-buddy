# Browser API

The browser API provides streaming conversion with Web Workers and WASM loading on demand.

## Minimal example

<SandpackExample
  template="vanilla-ts"
  activeFile="/index.ts"
  preview={false}
  files={{
    "/index.ts": `
import { convertToString } from "convert-buddy-js";

const output = await convertToString(file, { outputFormat: "json" });
`,
  }}
/>
